HEADINGS

Default stylings for HTML headings.

Default h1 size 2em

Default h2 size 1.5em

Default h3 size 1.17em

Default h4 size 1em

Default h5 size .83em
Default h6 size .67em

Nested <h1> elements

If an h1 has a parent element of aside, article, aside, or section, it will change the default font-size as shown below. Starts at 2em and steps down to .67em.

Default h1 style nested once 1.5em

Default h1 style nested twice 1.17em

Default h1 style nested thrice 1em

Default h1 style nested four times .83em

Default h1 style nested five times .67em
